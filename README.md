
# Hospitality Management Dashboard (Power BI)

## Project Overview

This project presents a data-driven analysis of the hospitality industry, focusing on hotel performance, occupancy trends, and revenue insights. The dashboard enables stakeholders to make informed business decisions through interactive visualizations.

---

## Business Problem

Hospitality businesses often face challenges in:

* Monitoring occupancy rates over time
* Identifying high-revenue segments
* Understanding seasonal demand patterns

This dashboard addresses these challenges by providing clear and interactive insights.

---

## Key Insights

* Peak occupancy is observed during weekends and holiday periods
* Luxury room categories contribute the highest share of revenue
* Off-season periods show a noticeable decline in bookings
* Certain locations consistently outperform others in revenue generation

---

## Features

* Interactive filters (date, location, room type)
* Revenue and occupancy trend analysis
* KPI indicators for performance tracking
* Drill-down capabilities for detailed analysis

---

## Tools and Technologies

* Power BI
* Excel / CSV dataset
* DAX (Data Analysis Expressions)
